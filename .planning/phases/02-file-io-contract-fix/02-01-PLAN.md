  <action>
1. Add imports: `import io`, `import shutil`, `import os` at top of file.

2. Create `RCloneFile(io.IOBase)` class ABOVE `RCloneFileSystem`. This class wraps a temp file with mode-specific behavior:

   - `__init__(self, fs, path, mode)`:
     - Store fs, path, mode.
     - Create temp dir via `tempfile.mkdtemp()`.
     - If `"r" in mode` (read): Eager download via `rclone.copy(fs._make_rclone_path(path), tmp_dir, show_progress=False)`. Check `Path(tmp_dir).iterdir()` for files. If no files found OR `RcloneException` caught, call `_cleanup()` and raise `FileNotFoundError(f"File not found: {path}")` with `from e` chaining if exception. Open the downloaded file with `builtins.open(files[0], mode)` and store as `self._f`.
     - If `"w" in mode` (write): Create temp file path as `Path(tmp_dir) / Path(path).name`. Open it with `builtins.open(tmp_path, mode)` and store as `self._f`. Store `self._tmp_file = tmp_path`.
   - Delegate `read`, `write`, `seek`, `tell` to `self._f`.
   - `readable()` returns `"r" in self.mode`, `writable()` returns `"w" in self.mode`, `seekable()` returns `True`.
   - `closed` property returns `self._closed` (init to `False`).
   - `close()`: If already closed, return. In try/finally: if write mode, close `self._f` first, then `rclone.copyto(self._tmp_file.as_posix(), self.fs._make_rclone_path(self.path), show_progress=False)`. Catch `RcloneException`, raise `OSError(f"Failed to upload {self.path}")` with `from e`. If read mode, just close `self._f`. In finally: set `self._closed = True`, call `self._cleanup()`, call `super().close()`.
   - `_cleanup(self)`: `shutil.rmtree(self._tmp_dir, ignore_errors=True)`.
   - `__del__(self)`: Safety net calling `self._cleanup()`.

3. On `RCloneFileSystem`, add `_open()` method:
   ```python
   def _open(self, path, mode="rb", block_size=None, autocommit=True,
             cache_options=None, **kwargs):
       """Return a file-like object for the given path."""
       if mode not in ("rb", "wb"):
           raise ValueError(
               f"Unsupported mode: {mode!r}. Supported modes: 'r', 'w', 'rb', 'wb'"
           )
       return RCloneFile(self, path, mode)
   ```

4. DELETE the entire existing `open()` method (lines 93-124, the `@contextlib.contextmanager` decorated method). Also remove the `import contextlib` since it will no longer be used.

5. Keep `builtins` import (still needed in RCloneFile). Remove `contextlib` import. The `Path` import is still needed for RCloneFile.

IMPORTANT: Always pass `show_progress=False` to all rclone calls. Use `rclone.copyto()` for writes (not `rclone.copy()`). The error message in `_open()` for unsupported modes should list all user-facing modes ('r', 'w', 'rb', 'wb') even though only binary modes reach `_open()`.
