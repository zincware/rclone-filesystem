   - Create bucket.
   - Write string with `"w"` mode, read back with `"r"` mode using same fs.
   - Assert roundtrip preserves content exactly.

4. `test_text_read_utf8(s3_base, fs_key, request)` -- parametrize:
   - Create bucket, put object with UTF-8 content `"Hallo Welt! Gruesse!".encode("utf-8")`.
   - Read with `"r"` mode, assert string equality.

5. `test_text_newline_handling(s3_base, fs_key, request)` -- parametrize:
   - Create bucket, put object with `b"line1\r\nline2\rline3\nline4"`.
   - Read with `"r"` mode, assert newlines are normalized (universal newline behavior).
